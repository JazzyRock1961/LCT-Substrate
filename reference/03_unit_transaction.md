# 03_unit_transaction

## Schematic
A state initialization resolves a split path trajectory by choosing between available coordinate options.

## Mechanics
* **Processor (if-then):** Evaluates nodes that possess alternative branches rather than static links.
* **Runtime/Stochastics (OR):** Forces the system to select an open route from an available set of arrays, simulating dynamic state tracking. Once a path is utilized, the remaining logic updates downstream targets until equilibrium is met.

## Expected Behavior
The terminal prints which dynamic route was pulled on Tick 1. The clock then advances the remaining open pathways and terminal nodes simultaneously until all transaction options match system parameters, halting the execution sequence.
